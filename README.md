# 中文关键词提取工具

这是一个基于jieba分词和多种算法的中文关键词提取工具。该工具提供了五种关键词提取方法：
1. 使用jieba内置的TF-IDF接口
2. 使用自定义实现的TF-IDF算法
3. 使用自定义实现的TextRank算法
4. 使用jieba内置的TextRank接口
5. 使用LDA（Latent Dirichlet Allocation）算法

## 功能特点

- 支持中文文本的关键词提取
- 提供五种关键词提取算法
- 支持自定义停用词
- 可配置返回的关键词数量
- 支持批量文本处理

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 基本用法：

```python
from keyword_extractor import KeywordExtractor

# 创建提取器实例
extractor = KeywordExtractor(topK=10)

# 使用jieba的TF-IDF接口
text = "你的文本内容"
keywords = extractor.extract_keywords_tfidf(text)

# 使用自定义TF-IDF实现
texts = ["文本1", "文本2", "文本3"]
keywords_list = extractor.extract_keywords_custom_tfidf(texts)

# 使用自定义TextRank实现
textrank_keywords = extractor.extract_keywords_textrank(text)

# 使用jieba的TextRank接口
jieba_textrank_keywords = extractor.extract_keywords_jieba_textrank(text)

# 使用LDA算法
lda_keywords = extractor.extract_keywords_lda(texts)
```

2. 添加停用词：
在 `data/stopwords.txt` 文件中添加停用词，每行一个词。

## 算法说明

### TF-IDF算法
- **原理**：基于词频-逆文档频率，衡量词语在文档中的重要程度
- **特点**：适合识别文档中的核心词汇，对高频词敏感

### TextRank算法
- **原理**：基于图排序算法，将词语作为节点，共现关系作为边
- **特点**：考虑词语间的语义关系，更适合提取语义相关的关键词

### LDA算法
- **原理**：基于主题模型，假设文档由多个主题组成，每个主题由词语分布表示
- **特点**：能够发现文档的潜在主题结构，适合大规模文档集合的关键词提取

### 算法对比

1. **TF-IDF方法**：更注重词频统计，适合识别高频关键词
2. **TextRank方法**：更注重词语间的语义关系，适合提取语义相关的关键词
3. **LDA方法**：能够发现文档的潜在主题结构，适合大规模文档集合
4. **jieba内置方法**：经过优化的实现，效果较好
5. **自定义实现**：提供更多自定义选项，适合特定需求

### LDA算法特点

- **主题发现**：能够自动发现文档集合中的潜在主题
- **语义关联**：基于主题模型，能够识别语义相关的关键词
- **大规模处理**：适合处理大规模文档集合
- **参数调优**：支持主题数量、训练轮数等参数调整

## 注意事项

- 确保文本编码为UTF-8
- 停用词文件（stopwords.txt）是可选的
- 自定义TF-IDF实现更适合批量处理多个文档
- TextRank算法需要足够的文本长度才能构建有效的词图
- LDA算法需要足够的文档数量才能训练有效的主题模型
